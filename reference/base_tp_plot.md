# base_tp_plot

Base time profile plot

## Usage

``` r
base_tp_plot(data, vpc_data, variable_names, variable_labels, use_tad = FALSE)
```

## Arguments

- data:

  A data.frame of data

- vpc_data:

  A data.frame of VPC data

- variable_names:

  A list of variable names

- variable_labels:

  A list of variable labels

- use_tad:

  Logical, if `TRUE`, use TAD instead of time
