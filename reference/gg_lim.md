# gg_lim

Apply `Min` and `Max` from `meta_data` as limits of the ggplot

## Usage

``` r
gg_lim(plot_object, meta_data, x = NULL, y = NULL)
```

## Arguments

- plot_object:

  A ggplot object

- meta_data:

  A meta_data data.frame

- x:

  A variable name for x-axis

- y:

  A variable name for y-axis
