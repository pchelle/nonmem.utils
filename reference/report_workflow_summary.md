# report_workflow_summary

Render a docx document that analyses the nonmem results

## Usage

``` r
report_workflow_summary(dir_path, report_path = "workflow-summary.docx")
```

## Arguments

- dir_path:

  Path to the nonmem results directory

- report_path:

  Path to the output report file
