# calc_tad

Calculate time after dose (TAD). Records before the first dose are
output \< 0. For infusions (`dur>0` or `rate>0`), the time after the end
of infusion is calculated.

## Usage

``` r
calc_tad(id, time, amt = NULL, evid = NULL, rate = NULL, dur = NULL)
```

## Arguments

- id:

  Subject identifiers values

- time:

  Time values

- amt:

  Amount values

- evid:

  Event identifiers values

- rate:

  Rate values

- dur:

  Duration values
